# StudyPet 🐱📚

A gamified Pomodoro study timer web app with a virtual pet companion.

## ✨ Features

- **Pomodoro Timer**: Customizable study and break intervals
- **Subject Tracking**: Color-coded subjects with detailed statistics
- **Coin Rewards**: Earn coins for completed study sessions
- **Pet Shop**: Purchase items, themes, and backgrounds
- **Study Statistics**: View your progress by week, month, or year
- **8 Color Themes**: Choose from 4 light and 4 dark themes
- **Interactive Pet**: Draggable and resizable cat character with blinking animations
- **Data Persistence**: All progress saved locally in your browser

## 🚀 Live Demo

**[Try StudyPet Live!](https://mystudypet.netlify.app/)** 🐱✨

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Touch & Gesture Events API
- Flaticon for UI icons

## 📦 Getting Started

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/KeybordKat/StudyPet.git
```

2. Open `index.html` in your browser:
```bash
cd StudyPet
open index.html
```

That's it! No build process or dependencies required.

## 📖 How to Use

1. Click the menu button (☰) in the top right
2. Select Timer to set up a study session
3. Choose a subject and customize your times
4. Start studying! Earn coins for each completed session
5. Visit the Shop to spend your coins on themes and items
6. Check Statistics to track your progress
