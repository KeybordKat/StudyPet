# StudyPet Development Roadmap 🗺️

This roadmap outlines the planned features and improvements for StudyPet.

---

## 🎯 Phase 1: User Authentication & Data Sync

### Login System
- [ ] Add "Sign in with Google" OAuth integration
- [ ] Add "Sign up with Email" option (email/password)
- [ ] Create user registration flow
- [ ] Build user profile page
- [ ] Enable cloud sync for user data (Firebase/Supabase)
- [ ] Migrate from LocalStorage to cloud database
- [ ] Allow users to access their data across devices
- [ ] Add "Continue as Guest" option for users who prefer local storage

### Security Features
- [ ] Implement secure password hashing and storage
- [ ] Add two-factor authentication (2FA) option
- [ ] Create session management and auto-logout
- [ ] Add password reset/recovery functionality
- [ ] Implement rate limiting for API requests
- [ ] Add HTTPS enforcement
- [ ] Create privacy policy and terms of service
- [ ] Add data export functionality (GDPR compliance)
- [ ] Implement data deletion option

---

## 📊 Phase 2: Advanced Analytics

### Enhanced Study Statistics
- [ ] Create detailed study patterns dashboard
- [ ] Add productivity heatmap (similar to GitHub contributions)
- [ ] Track study streaks and longest streak records
- [ ] Add time-of-day productivity analysis
- [ ] Create subject-wise performance comparison charts
- [ ] Add focus score calculation based on session completion rate
- [ ] Implement weekly/monthly study goals with progress tracking
- [ ] Add study efficiency metrics (actual vs planned time)
- [ ] Create exportable reports (PDF/CSV)
- [ ] Add year-in-review summary (annual statistics)

### Distraction Tracking
- [ ] Add "I Got Distracted" button during study sessions
- [ ] Track number of distractions per session
- [ ] Record distraction timestamps and duration
- [ ] Create distraction statistics dashboard
- [ ] Show distraction patterns by time/day/subject
- [ ] Add distraction categories (social media, phone, other)
- [ ] Calculate focus rate (study time vs distraction time)
- [ ] Set distraction reduction goals
- [ ] Add motivational messages to reduce distractions
- [ ] Create distraction history log
